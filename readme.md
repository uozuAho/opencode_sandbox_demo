# Opencode demo project

A toy coding project to demonstrate opencode config, agents etc.

Install
- [just](https://github.com/casey/just)
- [docker sandboxes](https://docs.docker.com/ai/sandboxes/)

then:

```sh
just -l
just run
just check-all
```
